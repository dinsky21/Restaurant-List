# restaurant_list
![餐廳清單]()
## 介紹
可利用關鍵字搜尋餐廳名稱及類別，點擊各餐廳可得知相關詳細資訊
## 功能
1. 在輸入框搜尋餐廳名稱或類別可篩選出相關餐廳
2. 點擊該餐廳可取得詳細資訊
## 開始使用
1. 請確認已安裝 node.js 與 npm
2. clone 此專案至本地
3. 在本地開啟之後，透過終端機進入資料夾:

   ```bash
   cd 資料夾名稱
   ```

   ```bash
   npm install
   ```

   ```bash
   nodemon app.js
   ```

4. 若看見此行訊息則代表順利運行，打開瀏覽器輸入以下網址

   ```bash
   The express is running on localhost:3000
   ```

## 開發工具
- node.js 16.15.0
- Express 4.16.4
- Express-Handlebars 6.0.6
- Bootstrap 5.2.0
